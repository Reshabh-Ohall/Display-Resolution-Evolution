# Evolution of Display Resolutions

| Era / Type          | Resolution (Scan Lines / Vertical) | Region / Notes                                    | Year / Approx               |       |
| ------------------- | ---------------------------------- | ------------------------------------------------- | --------------------------- | ----- |
| Mechanical TV       | 24                                 | Experimental                                      | 1920s                       |       |
| Mechanical TV       | 30                                 | John Logie Baird                                  | 1926                        |       |
| Mechanical TV       | 36                                 | USA experimental                                  | 1928                        |       |
| Mechanical TV       | 45                                 | USA experimental                                  | 1929                        |       |
| Mechanical TV       | 48                                 | Early experiments                                 | 1930                        |       |
| Mechanical TV       | 60                                 | Early experiments                                 | 1930-31                     |       |
| Early Electronic TV | 120                                | Experimental                                      | 1932                        |       |
| Early Electronic TV | 240                                | Hybrid mechanical/electronic                      | 1933                        |       |
| Early Electronic TV | 343                                | USSR / USA experiments                            | 1934                        |       |
| Early Electronic TV | 375                                | Germany / USA experimental                        | 1935                        |       |
| Electronic TV       | 405                                | UK standard, first practical electronic broadcast | 1936                        |       |
| Electronic TV       | 441                                | France, Germany, early Japan / USA                | Late 1930s                  |       |
| Electronic TV       | 450-455                            | Europe trials                                     | Late 1930s                  |       |
| Analog Broadcast    | 525                                | NTSC - USA, Japan                                 | 1941                        |       |
| Analog Broadcast    | 625                                | PAL / SECAM - Europe, India                       | 1949+                       |       |
| Analog Broadcast    | 819                                | France - high-resolution                          | 1949-1980s                  |       |
| Digital SD          | 480p                               | Standard Definition                               | 1990s                       |       |
| Digital SD          | 576p                               | Standard Definition PAL                           | 1990s                       |       |
| Digital HD          | 720p                               | High Definition                                   | 1998+                       |       |
| Digital Full HD     | 1080i / 1080p                      | Full HD                                           | 2000s                       |       |
| Digital QHD / 2K    | 1440p                              | WQHD / 2K displays                                | 2010s                       |       |
| Digital UHD         | 2160p                              | 4K Ultra HD                                       | 2010s                       |       |
| Digital UHD         | 4320p                              | 8K Ultra HD                                       | 2020s                       |       |
| Digital UHD         | 15360p                             | 16K Ultra HD                                      | Professional / Demo Screens | 2020s |

**Notes:**

* Early mechanical TVs had extremely low lines (24–60).
* 405–819 lines represent the golden age of analog broadcast.
* Post-819 lines, all higher resolutions required digital processing.
* Modern “p” values (e.g., 1080p, 4K, 16K) are pixel-based, not scan-line based.
